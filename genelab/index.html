Que es?

jDev GeneLab, también conocido como Project HEGADL (Human External Gene Aggregation Database and Laboratory) es un proyecto que intenta guardar perfiles de genes humanos externamente-visibles. Estos perfiles se agregarán y se guardarán en la base de datos pública HEGADL y se podrá experimentar en el laboratorio virtual HEGADL.

Cómo funciona?

jDev GeneLab te pedirá diferentes genes visibles para poder guardarte. Esto es completamente seguro, nadie te recococerá únicamente usando los datos que guardamos.

Privacidad

Nos importa mucho tu privacidad, el único número que guardamos que te identifica es el # de Sujeto. Tu nombre se guardará en la base de datos censurado, únicamente las iniciales se podrán leer. No queremos dinero o información personal. No la necesitamos. Queremos perfiles de genes exteriores, no queremos perfiles que te puedan identificar personalmente.
